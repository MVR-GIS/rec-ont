4.14 Property Matrix

Property Type Domain Range Role
recs:representsEntity Object property recs:RecreationSpatialFeature rec:RecreationDomainEntity Links a GIS representation to the managed recreation entity it represents.
recs:representsConcept Object property recs:RecreationSpatialFeature skos:Concept Links a GIS representation to a source-backed classification, zoning, or boundary concept.
rec:hasSpatialRepresentation Object property rec:RecreationDomainEntity recs:RecreationSpatialFeature Inverse of recs:representsEntity.
recs:associatedTrail Object property recs:TrailAssociatedFeature rec:Trail Links a trail-associated point feature to the trail it belongs with without claiming that the point feature is the trail line representation.
rec:hasTrailAssociatedFeature Object property rec:Trail recs:TrailAssociatedFeature Inverse of recs:associatedTrail.
recs:usesSpatialPattern Object property recs:RecreationSpatialFeature owl:Thing Declares whether the feature is modeled as a representation or direct feature pattern.
recs:intersectsPhysicalRealm Object property recs:RecreationSpatialFeature upr:PhysicalRealm Connects a spatial feature to land, water, or other physical realm context.
recs:sourceLayerName Datatype property recs:RecreationSpatialFeature xsd:string Preserves the source layer or feature class name.
recs:featureGrain Datatype property recs:RecreationSpatialFeature xsd:string Explains what one source row or feature represents.
recs:sourceIdentifier Datatype property recs:RecreationSpatialFeature xsd:string Preserves the source record identifier or crosswalk key.
